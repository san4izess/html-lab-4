# html lab4
Цей код є прикладом використання HTML і CSS для створення фонових зображень для абзаців, блоків і всього документа, а також використання різних рамок, включаючи рамки із зображенням і закруглені рамки. Код також містить приклад використання кількох фонових зображень одного блоку. Важливо помітити, що для вставки зображень використовувався тег <img> із зазначенням шляху до файлу зображення.
